## Funcionalidades JavaScript

- Sistema de **50 partículas** flotantes
- **Navbar sticky** con blur al hacer scroll
- **Menú hamburguesa** para mobile
- **Flip cards** con hover (desktop) y tap (mobile)
- **Contadores animados** con easing suave
- **Intersection Observer** para reveal de elementos al scroll
- **Smooth scroll** a secciones
- **Parallax suave** en el hero
- **Botón scroll-to-top**
- **Nav link activo** dinámico según sección visible
